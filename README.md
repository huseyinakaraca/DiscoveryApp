# DiscoveryApp

This is a cross-platform game discovery app I developed to practice .NET MAUI and API integration. It fetches game data using the RAWG API and uses SQLite for a local user authentication system.

## Features

* Fetch and search games via RAWG API.
* Local user registration and login using SQLite.
* Basic password validation.
* Save games to a local favorites list.

## Technologies Used

* .NET MAUI (C# & XAML)
* SQLite
* RAWG API
